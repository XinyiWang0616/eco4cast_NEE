# eco4cast_NEE
This repo is for Virginia tech eco4cast assignment. All datasets and baseline results are provided by eco4cast challenging catalog.

Target: daily Net ecosystem exchange (NEE)

Features: air_temperature, eastward_wind, surface_downwelling_shortwave_flux_in_air, precipitation_flux, relative_humidity, northward_wind, surface_downwelling_longwave_flux_in_air, air_pressure

Model: XGBoost
